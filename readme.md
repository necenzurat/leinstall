Web Server Tools (WST)
================

Web Server Tools, a collection of scripts to install and configure new vhosts, with nginx, php-fpm, mysql/percona


Usage
------------

**new.sh**

1. wget-it
2. run it
2.5 wait some minutes/seconds
3. enjoy your server


**server.sh**
